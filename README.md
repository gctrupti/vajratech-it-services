# VajraTech – IT Services Company Homepage

A modern, responsive homepage design for a professional IT services company.  
Built using React, Vite, and Tailwind CSS with a clean enterprise-style UI.

---

##  Project Overview

VajraTech is a fictional IT services company homepage designed to showcase:

- Strong hero section with clear CTA
- Service offerings
- Why Choose Us section
- Technologies & Partners showcase
- Client testimonials
- Conversion-focused CTA
- Professional footer

The design follows a light corporate blue theme suitable for enterprise IT consulting businesses.

---

##  Tech Stack

- React (Vite)
- Tailwind CSS
- Lucide React Icons

---

##  Features

- Fully responsive layout
- Professional enterprise UI
- Gradient-based brand styling
- Clean component-based architecture
- Modern hover effects and smooth transitions
- Structured folder organization
